===========================================
PROJET : SITE VITRINE - INSERTION PROFESSIONNELLE
===========================================

1. DESCRIPTION DU PROJET
-------------------------------------------
Ce projet est un site vitrine réalisé en HTML, CSS et JavaScript.
Il présente une plateforme d’accompagnement à l’insertion professionnelle.

Le site a pour objectif de :
- Informer les utilisateurs sur les services proposés
- Présenter un parcours d’accompagnement
- Permettre la prise de contact

-------------------------------------------

2. STRUCTURE DU PROJET
-------------------------------------------
Le projet est organisé comme suit :

- index.html       → Page d’accueil
- apropos.html     → Page à propos
- services.html    → Page des services
- contact.html     → Page de contact

- css/
    └── style.css  → Fichier de style principal

- js/
    └── script.js  → Script pour animations et interactions

- images/
    └── (contient les images du site)

-------------------------------------------

3. FONCTIONNALITÉS
-------------------------------------------
- Navigation entre plusieurs pages
- Design moderne et responsive (adapté mobile)
- Animations au scroll
- Formulaire de contact (simulation avec JavaScript)
- Sections :
  - Accueil
  - À propos
  - Services
  - Parcours
  - Contact

-------------------------------------------

4. UTILISATION DU SITE
-------------------------------------------
Pour lancer le site :

1. Ouvrir le dossier du projet
2. Double-cliquer sur :
   index.html

OU

Ouvrir avec un navigateur :
- Google Chrome
- Firefox
- Edge

-------------------------------------------

5. IMAGES
-------------------------------------------
Les images utilisées sont stockées dans le dossier :

images/

IMPORTANT :
Les noms des images doivent correspondre à ceux utilisés dans le CSS.

Exemple :
- photo-accueil.jpg
- photo-apropos.jpg
- photo-services.jpg
- photo-contact.jpg

-------------------------------------------

6. TECHNOLOGIES UTILISÉES
-------------------------------------------
- HTML5 → structure du site
- CSS3 → design et mise en page
- JavaScript → interactions et animations

-------------------------------------------
